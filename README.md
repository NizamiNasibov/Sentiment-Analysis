# Sentiment-Analysis
Sentiment Analysis üzrə öyrənilən texnikalar və həll yolları

Araşdırma zamanı mətnlərin emosional tonunun müəyyən edilməsi üçün əsas yanaşmalar öyrənilmişdir. İlk mərhələdə mətnlərin əvvəlcədən emalı aparılır: lazımsız simvolların silinməsi, sözlərin normallaşdırılması və stop-word-lərin çıxarılması.

Sentiment analizi üçün lüğət əsaslı metodlardan istifadə olunur ki, bu yanaşma sadə və sürətlidir, lakin konteksti tam əhatə etmir. Daha dəqiq nəticə əldə etmək üçün maşın öyrənməsi üsulları tətbiq edilir; bu zaman mətnlər TF-IDF kimi metodlarla ədədi formaya salınır və klassifikasiya modelləri qurulur.

Müasir yanaşma olaraq BERT kimi dərin öyrənmə modelləri kontekstin daha düzgün anlaşılmasını təmin edir. Modelin effektivliyi accuracy və F1-score kimi metrikalarla qiymətləndirilir.

Bu texnikalar müştəri rəylərinin, sosial media mətnlərinin və istifadəçi müraciətlərinin analizi üçün istifadə olunur.
